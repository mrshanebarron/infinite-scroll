# Laravel Design Infinite Scroll

Infinite scroll component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/infinite-scroll
```

## Usage

### Livewire Component

```blade
<livewire:ld-infinite-scroll />
```

### Blade Component

```blade
<x-ld-infinite-scroll />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-infinite-scroll-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-infinite-scroll-views
```
